# Hooks — Custom React Hooks

| Hook | Description |
|------|-------------|
| `useAsk.ts` | Query state management: submit, loading, error, reset |
| `useScrollReveal.ts` | Intersection Observer-based scroll animation trigger |
| `useVoiceAssistant.ts` | Voice input (Speech-to-Text) and output via modular TTS (`lib/tts/`) with continuous Siri-like voice mode |
