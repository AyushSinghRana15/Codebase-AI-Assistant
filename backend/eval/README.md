# Eval — Evaluation Harness

Automated RAGAS-based evaluation for measuring pipeline quality.

| File | Description |
|------|-------------|
| `ragas_eval.py` | RAGAS metric computation (faithfulness, relevancy, precision, recall) |
| `run_eval.py` | Script to run full evaluation suite |
| `scorecard.md` | Historical evaluation scores |
| `test_queries.json` | Test query dataset |
